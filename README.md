# Jumping Game (Geometry Dash-style)

A browser Geometry Dash-style runner with **20 levels**, jump pads, spikes, and unlock progression.

## Play

Open `index.html` in your browser, or play on GitHub Pages after it's enabled:

**https://lucas206201.github.io/JumpingGame/**

## Controls

- **Click / Tap / Space** — jump
- **Esc** — level select

## Levels

Clear a level to unlock the next. Progress is saved in your browser.
